AVL Dictionary

Dictionaries map words to their definition. So, a dictionary can be implemented by maping two sets A and B where the input member A is mapped to the output member B. This mapping can be done by a searching algorithm. AVL tree which is a self-balancing binary search tree, is used as the searching algorithm in this project. The searching algorithm matches the output member B for the given input member A.
